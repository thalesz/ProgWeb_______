A Linguagem HTML

HTML(abreviação para a expressão inglesa HyperText arkup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores

Alguns dos recursos da linguagem HTML vistos até este momento:

Recrusos Tags Associadas Descrição
Links < a >

 Criar links entre as páginas
Listas < ul > e < li >

 Criar lista de itens ordenados
< ol > e < li >

 Criar lista de itens nao ordenados
Imagens < img >

 Inserir imagens
